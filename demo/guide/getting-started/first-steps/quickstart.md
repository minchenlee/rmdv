# Quickstart

[← back to README](../../../README.md)

rmdv opens a folder as a *vault* and renders any markdown file you click.

## Core shortcuts

| Key | Action |
|-----|--------|
| ⌘O | Open file or folder |
| ⌘B | Toggle sidebar |
| ⌘M | Mind map of the current doc |
| ⌘E | Edit the current file |
| ⌘⇧F | Search the whole vault |
| ⌘↑ / ⌘↓ | Previous / next heading |
| ⌘= / ⌘− | Zoom text in / out |

## Navigating a vault

1. Open a folder (this `demo/`).
2. The sidebar shows the tree; the breadcrumb shows where you are.
3. Click any `.md` to render it.
   - Internal links like [the syntax page](../../features/markdown/syntax.md) jump in place.
   - External links open in your browser.
4. Press **⌘M** to see the document as a mind map.

> **Note**
> rmdv resolves relative links against the *current file's* folder — so a vault
> can cross-link freely across deep directories.
