# iced_math v0.1 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship `iced_math` v0.1.0 — a pure-Rust, zero-JS, native Iced 0.14 widget that renders a Tier 1 subset of LaTeX math (~50 commands: atoms, frac, sub/sup, sqrt, basic ops, parens, greek) via SVG path output.

**Architecture:** New standalone crate in a **separate repo** from rmdv. Pipeline: `pulldown-latex` parser events → IR tree → TeX-style boxer (uses OpenType MATH table via `ttf-parser`) → SVG bytes (glyph outlines as `<path>`, rules as `<rect>`) → consumed by `iced::widget::svg`. No custom `Widget` impl. No JavaScript. Glyphs identified by `GlyphId` so MATH-table variants and `GlyphAssembly` extensible glyphs are addressable (Iced's text shaper can't address arbitrary glyph IDs).

**Tech Stack:** Rust 1.75+ (MSRV), Iced 0.14 (features: `svg`, `advanced`), pulldown-latex 0.7, ttf-parser 0.25. Dev: insta 1, resvg 0.46, tiny-skia 0.11, criterion 0.5. Bundled font: Latin Modern Math OTF (fallback STIX Two Math if GUST license blocks redistribution).

**Spec:** `docs/superpowers/specs/2026-05-23-iced-math-design.md` (lives in rmdv repo for reference; iced_math repo gets its own copy under `docs/`).

**Repo:** Separate GitHub repo `iced_math`. Implementation does NOT touch rmdv source. rmdv integration is a separate later plan (Task 25, deferred).

**v0.1 scope (from spec §12 roadmap):**
- LaTeX atoms: letters, numbers, operators `+ − × ÷ = < > ≤ ≥`
- Greek letters (lowercase + uppercase)
- Super/subscripts (`x^2`, `a_i`, `x^{n+1}`, `a_i^j`)
- Fractions (`\frac{a}{b}`, `\dfrac`, `\tfrac`)
- Square roots (`\sqrt{x}`, `\sqrt[n]{x}`)
- Basic operators with limits (sum/int/prod in display mode get limits above/below; text mode = scripts)
- Parentheses, brackets, braces — `(`, `[`, `{`, with `\left ... \right` auto-sizing via MATH variants (NOT GlyphAssembly — that's v0.2)
- Display vs inline distinction
- Error fallback: red monospace source

**v0.1 NON-goals (deferred):**
- Matrices, aligned, cases, multiline environments → v0.2
- Accents (`\hat`, `\bar`, etc.) → v0.2
- AMS symbols, color, sizing modes → v0.3
- `GlyphAssembly` for delimiters taller than tallest variant → v0.2
- `Math::parse` cached handle, themed color override → v0.5

/// Pixel-space metrics for a glyph at a given font size.
/// All values are in SVG-down y space; `height` is above baseline, `depth` below.
#[derive(Debug, Clone, Copy)]
pub struct GlyphMetrics {
    pub advance: f32,
    pub height: f32,
    pub depth: f32,
}

pub fn glyph_metrics(id: GlyphId, font_size: f32) -> GlyphMetrics {
    let face = face();
    let upem = face.units_per_em() as f32;
    let scale = font_size / upem;

    let advance = face.glyph_hor_advance(id).unwrap_or(0) as f32 * scale;
    let bbox = face.glyph_bounding_box(id);
    let (height, depth) = match bbox {
        // ttf-parser bbox is y-up: y_max above baseline, y_min below (negative)
        Some(b) => (b.y_max as f32 * scale, (-(b.y_min as f32)) * scale),
        None => (0.0, 0.0),
    };
    GlyphMetrics { advance, height, depth }
}
```

use std::fmt::Write;

struct PathBuilder(String);

impl ttf_parser::OutlineBuilder for PathBuilder {
    fn move_to(&mut self, x: f32, y: f32) {
        let _ = write!(self.0, "M{} {} ", x, y);
    }
    fn line_to(&mut self, x: f32, y: f32) {
        let _ = write!(self.0, "L{} {} ", x, y);
    }
    fn quad_to(&mut self, x1: f32, y1: f32, x: f32, y: f32) {
        let _ = write!(self.0, "Q{} {} {} {} ", x1, y1, x, y);
    }
    fn curve_to(&mut self, x1: f32, y1: f32, x2: f32, y2: f32, x: f32, y: f32) {
        let _ = write!(self.0, "C{} {} {} {} {} {} ", x1, y1, x2, y2, x, y);
    }
    fn close(&mut self) {
        self.0.push_str("Z ");
    }
}

/// Emit the glyph's outline as an SVG path data string in font design units (y-up).
/// Caller MUST apply `matrix(s 0 0 -s ox oy)` transform where `s = font_size / units_per_em`
/// to convert to SVG (y-down) pixel space.
/// Returns empty string for blank glyphs.
pub fn outline_path(id: GlyphId) -> String {
    let mut b = PathBuilder(String::new());
    let _ = face().outline_glyph(id, &mut b);
    b.0
}
```

/// Subset of OpenType MATH table constants needed for v0.1 layout.
/// Full list in OpenType spec §6.4.2 — add variants as later tiers need them.
#[derive(Debug, Clone, Copy)]
pub enum MathConstant {
    AxisHeight,
    FractionNumeratorShiftUp,
    FractionDenominatorShiftDown,
    FractionRuleThickness,
    FractionNumDisplayStyleShiftUp,
    FractionDenomDisplayStyleShiftDown,
    SubscriptShiftDown,
    SubscriptTopMax,
    SubscriptBaselineDropMin,
    SuperscriptShiftUp,
    SuperscriptShiftUpCramped,
    SuperscriptBottomMin,
    SuperscriptBaselineDropMax,
    SubSuperscriptGapMin,
    SuperscriptBottomMaxWithSubscript,
    RadicalRuleThickness,
    RadicalVerticalGap,
    RadicalDisplayStyleVerticalGap,
    RadicalKernBeforeDegree,
    RadicalKernAfterDegree,
    RadicalDegreeBottomRaisePercent,
    UpperLimitGapMin,
    UpperLimitBaselineRiseMin,
    LowerLimitGapMin,
    LowerLimitBaselineDropMin,
}

/// Look up a MATH constant in pixel space at `font_size`. Returns 0.0 if the
/// font has no MATH table or the constant is absent.
pub fn math_constant(c: MathConstant, font_size: f32) -> f32 {
    let face = face();
    let scale = font_size / face.units_per_em() as f32;
    let Some(math) = face.tables().math else { return 0.0 };
    let consts = math.constants;
    use MathConstant::*;
    let value = match c {
        AxisHeight => consts.axis_height().value,
        FractionNumeratorShiftUp => consts.fraction_numerator_shift_up().value,
        FractionDenominatorShiftDown => consts.fraction_denominator_shift_down().value,
        FractionRuleThickness => consts.fraction_rule_thickness().value,
        FractionNumDisplayStyleShiftUp => consts.fraction_numerator_display_style_shift_up().value,
        FractionDenomDisplayStyleShiftDown => consts.fraction_denominator_display_style_shift_down().value,
        SubscriptShiftDown => consts.subscript_shift_down().value,
        SubscriptTopMax => consts.subscript_top_max().value,
        SubscriptBaselineDropMin => consts.subscript_baseline_drop_min().value,
        SuperscriptShiftUp => consts.superscript_shift_up().value,
        SuperscriptShiftUpCramped => consts.superscript_shift_up_cramped().value,
        SuperscriptBottomMin => consts.superscript_bottom_min().value,
        SuperscriptBaselineDropMax => consts.superscript_baseline_drop_max().value,
        SubSuperscriptGapMin => consts.sub_superscript_gap_min().value,
        SuperscriptBottomMaxWithSubscript => consts.superscript_bottom_max_with_subscript().value,
        RadicalRuleThickness => consts.radical_rule_thickness().value,
        RadicalVerticalGap => consts.radical_vertical_gap().value,
        RadicalDisplayStyleVerticalGap => consts.radical_display_style_vertical_gap().value,
        RadicalKernBeforeDegree => consts.radical_kern_before_degree().value,
        RadicalKernAfterDegree => consts.radical_kern_after_degree().value,
        RadicalDegreeBottomRaisePercent => return consts.radical_degree_bottom_raise_percent() as f32 / 100.0,
        UpperLimitGapMin => consts.upper_limit_gap_min().value,
        UpperLimitBaselineRiseMin => consts.upper_limit_baseline_rise_min().value,
        LowerLimitGapMin => consts.lower_limit_gap_min().value,
        LowerLimitBaselineDropMin => consts.lower_limit_baseline_drop_min().value,
    };
    value as f32 * scale
}
```

NOTE: exact method names on `ttf_parser`'s MATH table may differ from this draft — the implementer must verify against `ttf_parser` 0.25 docs at first build and pin the actual accessor names. If methods do not exist, fall back to reading `consts.<field>` directly; the field names match the OpenType spec.

```rust
//! pulldown-latex events → IR tree.

use pulldown_latex::{Parser, event::Event, Storage};

use crate::font;
use crate::ir::{Node, AtomClass, Style};

#[derive(Debug)]
pub struct ParseError(pub String);

pub fn to_ir(src: &str, font_size: f32, style: Style) -> Result<Node, ParseError> {
    let storage = Storage::new();
    let parser = Parser::new(src, &storage);

    let mut row = Vec::new();
    for ev in parser {
        let ev = ev.map_err(|e| ParseError(format!("{e:?}")))?;
        handle_event(ev, &mut row, font_size, style)?;
    }
    Ok(Node::Row(row))
}

fn handle_event(
    ev: Event,
    out: &mut Vec<Node>,
    font_size: f32,
    style: Style,
) -> Result<(), ParseError> {
    use pulldown_latex::event::*;
    let size = style.font_size(font_size);
    match ev {
        Event::Content(Content::Ordinary { content, .. }) |
        Event::Content(Content::Number(content)) => {
            for ch in content.chars() {
                let class = classify(ch);
                let glyph = font::glyph_id(ch).ok_or_else(|| ParseError(format!("no glyph for {:?}", ch)))?;
                out.push(Node::Atom { class, glyph, font_size: size });
            }
        }
        Event::Content(Content::BinaryOp(op)) => {
            let ch = op.content.chars().next().ok_or_else(|| ParseError("empty op".into()))?;
            let glyph = font::glyph_id(ch).ok_or_else(|| ParseError(format!("no glyph for {:?}", ch)))?;
            out.push(Node::Atom { class: AtomClass::Bin, glyph, font_size: size });
        }
        Event::Content(Content::Relation(rel)) => {
            let ch = rel.content.chars().next().ok_or_else(|| ParseError("empty rel".into()))?;
            let glyph = font::glyph_id(ch).ok_or_else(|| ParseError(format!("no glyph for {:?}", ch)))?;
            out.push(Node::Atom { class: AtomClass::Rel, glyph, font_size: size });
        }
        // Other event variants handled in later tasks. For now skip unknown.
        _ => {}
    }
    Ok(())
}

fn classify(ch: char) -> AtomClass {
    match ch {
        '+' | '−' | '-' | '×' | '*' | '÷' | '/' | '±' | '∓' => AtomClass::Bin,
        '=' | '<' | '>' | '≤' | '≥' | '≠' | '∈' | '∉' | '⊂' | '⊃' | '≈' | '≡' => AtomClass::Rel,
        '(' | '[' | '{' => AtomClass::Open,
        ')' | ']' | '}' => AtomClass::Close,
        ',' | ';' | '.' | ':' => AtomClass::Punct,
        _ => AtomClass::Ord,
    }
}
```

NOTE: `pulldown-latex` 0.7 event/content variant names must be verified against actual docs at first build — the names above are the spec's *intent*. If the real API uses different variant names (e.g., `Event::Content(Content::Char(...))` instead of `Ordinary`), adjust accordingly.

In `handle_event`, add cases for `Event::BeginGroup`, `Event::EndGroup`, `Event::Visual(Visual::Subscript)`, `Event::Visual(Visual::Superscript)`.

Strategy: maintain a stack of in-progress rows. When a `Subscript`/`Superscript` event arrives, attach to the most recent atom on the row, wrapping it in `Subsup`. Braced groups push a new row, pop on `EndGroup`.

Pseudocode (adjust to real pulldown-latex API):

```rust
struct ParserState {
    stack: Vec<Vec<Node>>,
    pending_attach: Option<Attach>,    // sub or sup awaiting next atom/group
}

```

When event is `Subscript`: set `pending_attach = Some(Sub)`. Next atom or group, instead of pushing as `Node::Atom`, look at the last node on the current row, wrap it as `Subsup { base: last, sub: Some(this), sup: None }`. If last is already `Subsup`, mutate its `sub` slot.

Full implementation in this step:

```rust
// Replace to_ir/handle_event with a state-machine version.

use std::mem;

pub fn to_ir(src: &str, font_size: f32, style: Style) -> Result<Node, ParseError> {
    let storage = Storage::new();
    let parser = Parser::new(src, &storage);
    let mut state = ParserState::new(font_size, style);
    for ev in parser {
        let ev = ev.map_err(|e| ParseError(format!("{e:?}")))?;
        state.feed(ev)?;
    }
    state.finish()
}

struct ParserState {
    stack: Vec<Vec<Node>>,
    pending: Option<Attach>,
    font_size: f32,
    style: Style,
}

#[derive(Clone, Copy)]
enum Attach { Sub, Sup }

impl ParserState {
    fn new(font_size: f32, style: Style) -> Self {
        Self { stack: vec![Vec::new()], pending: None, font_size, style }
    }

    fn push_node(&mut self, node: Node) {
        if let Some(att) = self.pending.take() {
            // Attach to last atom on top row
            let row = self.stack.last_mut().expect("stack non-empty");
            let base = row.pop().unwrap_or(Node::Row(Vec::new()));
            let merged = match base {
                Node::Subsup { base, sub, sup } => {
                    let (sub, sup) = match att {
                        Attach::Sub => (Some(Box::new(node)), sup),
                        Attach::Sup => (sub, Some(Box::new(node))),
                    };
                    Node::Subsup { base, sub, sup }
                }
                other => {
                    let (sub, sup) = match att {
                        Attach::Sub => (Some(Box::new(node)), None),
                        Attach::Sup => (None, Some(Box::new(node))),
                    };
                    Node::Subsup { base: Box::new(other), sub, sup }
                }
            };
            row.push(merged);
        } else {
            self.stack.last_mut().unwrap().push(node);
        }
    }

    fn feed(&mut self, ev: Event) -> Result<(), ParseError> {
        use pulldown_latex::event::*;
        let size = self.style.font_size(self.font_size);
        match ev {
            Event::BeginGroup => self.stack.push(Vec::new()),
            Event::EndGroup => {
                let row = self.stack.pop().ok_or_else(|| ParseError("unmatched }".into()))?;
                self.push_node(Node::Row(row));
            }
            Event::Visual(Visual::Subscript) => self.pending = Some(Attach::Sub),
            Event::Visual(Visual::Superscript) => self.pending = Some(Attach::Sup),
            Event::Content(Content::Ordinary { content, .. }) | Event::Content(Content::Number(content)) => {
                for ch in content.chars() {
                    let class = classify(ch);
                    let glyph = font::glyph_id(ch).ok_or_else(|| ParseError(format!("no glyph for {:?}", ch)))?;
                    self.push_node(Node::Atom { class, glyph, font_size: size });
                }
            }
            Event::Content(Content::BinaryOp(op)) => {
                let ch = op.content.chars().next().ok_or_else(|| ParseError("empty op".into()))?;
                let glyph = font::glyph_id(ch).ok_or_else(|| ParseError(format!("no glyph for {:?}", ch)))?;
                self.push_node(Node::Atom { class: AtomClass::Bin, glyph, font_size: size });
            }
            Event::Content(Content::Relation(rel)) => {
                let ch = rel.content.chars().next().ok_or_else(|| ParseError("empty rel".into()))?;
                let glyph = font::glyph_id(ch).ok_or_else(|| ParseError(format!("no glyph for {:?}", ch)))?;
                self.push_node(Node::Atom { class: AtomClass::Rel, glyph, font_size: size });
            }
            _ => {}
        }
        Ok(())
    }

    fn finish(mut self) -> Result<Node, ParseError> {
        if self.stack.len() != 1 { return Err(ParseError("unclosed group".into())); }
        Ok(Node::Row(self.stack.pop().unwrap()))
    }
}
```

//! TeX inter-atom spacing table (TeXbook p. 170).

use crate::ir::AtomClass;

#[derive(Debug, Clone, Copy, PartialEq, Eq)]
pub enum Spacing {
    None,
    Thin,    // 3mu
    Med,     // 4mu
    Thick,   // 5mu
}

impl Spacing {
    /// Spacing width in pixels at the given font size. `mu` = 1/18 em.
    pub fn to_px(self, font_size: f32) -> f32 {
        let em = font_size;
        match self {
            Spacing::None => 0.0,
            Spacing::Thin => em * 3.0 / 18.0,
            Spacing::Med => em * 4.0 / 18.0,
            Spacing::Thick => em * 5.0 / 18.0,
        }
    }
}

/// Spacing between two adjacent atoms.
/// `display_or_text` = true if current style is Display or Text (not Script/ScriptScript);
/// Med and Thick spacings are suppressed in script styles per TeX rules.
pub fn between(left: AtomClass, right: AtomClass, display_or_text: bool) -> Spacing {
    use AtomClass::*;
    let raw = match (left, right) {
        (Ord, Op) | (Op, Ord) | (Op, Op) => Spacing::Thin,
        (Ord, Bin) | (Bin, Ord) | (Bin, Open) | (Close, Bin) | (Inner, Bin) | (Bin, Inner) => Spacing::Med,
        (Ord, Rel) | (Rel, Ord) | (Close, Rel) | (Rel, Open) | (Inner, Rel) | (Rel, Inner) => Spacing::Thick,
        (Ord, Inner) | (Inner, Ord) | (Op, Inner) | (Inner, Op) | (Close, Inner) | (Inner, Close)
        | (Inner, Open) | (Punct, _) | (Inner, Inner) | (Ord, Op) => Spacing::Thin,
        _ => Spacing::None,
    };
    if !display_or_text && (raw == Spacing::Med || raw == Spacing::Thick) {
        Spacing::None
    } else {
        raw
    }
}

//! IR → positioned Box tree.

use crate::font;
use crate::ir::{Node, Style};

#[derive(Debug, Clone)]
pub struct Box {
    pub width: f32,
    pub height: f32,
    pub depth: f32,
    pub kind: BoxKind,
}

#[derive(Debug, Clone)]
pub enum BoxKind {
    Glyph { glyph_id: ttf_parser::GlyphId, font_size: f32 },
    HBox(Vec<Child>),
    VBox(Vec<Child>),
    Rule { thickness: f32 },
    Empty,
}

#[derive(Debug, Clone)]
pub struct Child {
    pub offset: Point,
    pub child: Box,
}

#[derive(Debug, Clone, Copy)]
pub struct Point { pub x: f32, pub y: f32 }

pub fn layout(node: &Node, style: Style) -> Box {
    match node {
        Node::Atom { glyph, font_size, .. } => {
            let m = font::glyph_metrics(*glyph, *font_size);
            Box {
                width: m.advance,
                height: m.height,
                depth: m.depth,
                kind: BoxKind::Glyph { glyph_id: *glyph, font_size: *font_size },
            }
        }
        Node::Row(items) => layout_row(items, style),
        Node::Error(_) => Box { width: 0.0, height: 0.0, depth: 0.0, kind: BoxKind::Empty },
        // Frac, Subsup, Radical, Fenced, Op, Space handled in later tasks.
        _ => Box { width: 0.0, height: 0.0, depth: 0.0, kind: BoxKind::Empty },
    }
}

fn layout_row(items: &[Node], style: Style) -> Box {
    use crate::spacing;
    let mut children = Vec::new();
    let mut cursor = 0.0_f32;
    let mut height: f32 = 0.0;
    let mut depth: f32 = 0.0;

    let display_or_text = matches!(style, Style::Display | Style::Text);

    let mut prev_class: Option<crate::ir::AtomClass> = None;
    for node in items {
        let cur_class = atom_class(node);
        if let (Some(pc), Some(cc)) = (prev_class, cur_class) {
            let sp = spacing::between(pc, cc, display_or_text);
            cursor += sp.to_px(approx_font_size(node));
        }
        let b = layout(node, style);
        height = height.max(b.height);
        depth = depth.max(b.depth);
        let width = b.width;
        children.push(Child { offset: Point { x: cursor, y: 0.0 }, child: b });
        cursor += width;
        prev_class = cur_class;
    }
    Box { width: cursor, height, depth, kind: BoxKind::HBox(children) }
}

fn atom_class(node: &Node) -> Option<crate::ir::AtomClass> {
    match node {
        Node::Atom { class, .. } => Some(*class),
        Node::Op { .. } => Some(crate::ir::AtomClass::Op),
        Node::Fenced { .. } => Some(crate::ir::AtomClass::Inner),
        Node::Frac { .. } | Node::Radical { .. } | Node::Subsup { .. } | Node::Row(_) => Some(crate::ir::AtomClass::Ord),
        _ => None,
    }
}

fn approx_font_size(node: &Node) -> f32 {
    match node {
        Node::Atom { font_size, .. } | Node::Op { font_size, .. } => *font_size,
        _ => 16.0,
    }
}
```

fn layout_frac(num: &Node, den: &Node, style: Style) -> Box {
    use crate::font::{math_constant, MathConstant};
    let num_style = match style { Style::Display => Style::Text, _ => style.sub() };
    let den_style = num_style;     // denominator same script-step
    let n = layout(num, num_style);
    let d = layout(den, den_style);

    let base = approx_font_size_from_box(&n).max(approx_font_size_from_box(&d));
    let rule_thickness = math_constant(MathConstant::FractionRuleThickness, base).max(0.5);
    let axis = math_constant(MathConstant::AxisHeight, base);

    let (shift_up, shift_down) = if style.is_display() {
        (
            math_constant(MathConstant::FractionNumDisplayStyleShiftUp, base),
            math_constant(MathConstant::FractionDenomDisplayStyleShiftDown, base),
        )
    } else {
        (
            math_constant(MathConstant::FractionNumeratorShiftUp, base),
            math_constant(MathConstant::FractionDenominatorShiftDown, base),
        )
    };

    let width = n.width.max(d.width);
    let num_x = (width - n.width) / 2.0;
    let den_x = (width - d.width) / 2.0;

    // SVG y-down: numerator above axis (smaller y), denominator below axis (larger y).
    // Axis is at y = baseline - axis_height; place baseline of frac box wherever caller wants.
    // We build with baseline at y=0 (numerator ascends to negative y? No — boxer uses y-down
    // SVG space where positive y goes down from baseline. We store positions relative to
    // baseline, with positive y = below.)
    //
    // Standard convention: HBox/VBox child offsets are in SVG-down coordinates where
    // (0,0) is the baseline-top-left of the containing box.
    //
    // Numerator child baseline sits at y = -shift_up + axis (above container baseline).
    // Denominator baseline sits at y = +shift_down + axis.
    //
    // For child offsets, we want the child's *top-left* corner. Child's baseline is at
    // child.height below its top, so:
    //   num.offset.y = -(shift_up + axis) + n.height? Need to be precise.
    //
    // We'll define: box baseline is at y = box.height (i.e. depth grows below baseline,
    // height extends above baseline, top-left at y=0). For a child Box at offset (cx, cy),
    // cy is the y-coordinate of the *child's top* inside the parent.
    //
    // Numerator: its baseline must sit at parent_axis - shift_up.
    //   child top in parent = parent_axis - shift_up - n.height
    //   parent_axis = parent.height + (-axis)? No — axis is offset above baseline.
    //   Convention: in math, axis is at y = baseline_y - axis_height (above baseline).
    //   With "top-left = (0,0)" coords, parent baseline is at y = parent.height,
    //   axis at y = parent.height - axis.
    //
    // Compute parent dimensions first:
    let parent_height = n.height + n.depth + shift_up - axis;
    let parent_depth = d.height + d.depth + shift_down + axis;

    let num_top = parent_height - shift_up - axis - n.height;
    let den_top = parent_height - axis + shift_down;

    // Rule sits at axis line, centered on it
    let rule_top = parent_height - axis - rule_thickness / 2.0;

    let children = vec![
        Child { offset: Point { x: num_x, y: num_top }, child: n },
        Child { offset: Point { x: 0.0, y: rule_top }, child: Box {
            width, height: rule_thickness, depth: 0.0, kind: BoxKind::Rule { thickness: rule_thickness },
        }},
        Child { offset: Point { x: den_x, y: den_top }, child: d },
    ];

    Box {
        width,
        height: parent_height,
        depth: parent_depth,
        kind: BoxKind::VBox(children),
    }
}

fn approx_font_size_from_box(b: &Box) -> f32 {
    match &b.kind {
        BoxKind::Glyph { font_size, .. } => *font_size,
        BoxKind::HBox(c) | BoxKind::VBox(c) => c.first().map(|x| approx_font_size_from_box(&x.child)).unwrap_or(16.0),
        _ => 16.0,
    }
}

If the y-direction conventions are off, adjust here — boxer.rs is the single source of truth for coord conventions and downstream svg.rs (Task 19) inherits them.

fn layout_subsup(base: &Node, sub: Option<&Node>, sup: Option<&Node>, style: Style) -> Box {
    use crate::font::{math_constant, MathConstant};
    let b = layout(base, style);
    let script_style = style.sub();
    let s_sub = sub.map(|n| layout(n, script_style));
    let s_sup = sup.map(|n| layout(n, script_style));

    let base_size = approx_font_size_from_box(&b);
    let sup_shift_up = math_constant(MathConstant::SuperscriptShiftUp, base_size);
    let sub_shift_down = math_constant(MathConstant::SubscriptShiftDown, base_size);

    // Decide which children exist; place each relative to base baseline
    let mut max_w = b.width;
    let mut height = b.height;
    let mut depth = b.depth;

    let mut children = vec![Child { offset: Point { x: 0.0, y: 0.0 /*placeholder*/}, child: b.clone() }];

    if let Some(sup_box) = &s_sup {
        let sup_baseline_y = -sup_shift_up;  // baseline above base baseline by sup_shift_up
        let sup_top = sup_baseline_y - sup_box.height;
        max_w = max_w.max(b.width + sup_box.width);
        height = height.max(-sup_top);   // positive height grows up
        // ... (place sup at x = b.width, y = baseline_offset + sup_top translated to top-left coords)
    }
    if let Some(sub_box) = &s_sub {
        let sub_baseline_y = sub_shift_down;
        let sub_top = sub_baseline_y - sub_box.height;
        let _ = sub_box.width;
        depth = depth.max(sub_baseline_y + sub_box.depth);
    }

    // Recompute child positions with parent height as baseline reference.
    let parent_height = height;
    let parent_depth = depth;

    let mut final_children = Vec::new();
    final_children.push(Child {
        offset: Point { x: 0.0, y: parent_height - b.height },
        child: b.clone(),
    });
    if let Some(sup_box) = s_sup {
        let sup_baseline_in_parent = parent_height - sup_shift_up;
        let sup_top = sup_baseline_in_parent - sup_box.height;
        final_children.push(Child {
            offset: Point { x: b.width, y: sup_top },
            child: sup_box,
        });
    }
    if let Some(sub_box) = s_sub {
        let sub_baseline_in_parent = parent_height + sub_shift_down;
        let sub_top = sub_baseline_in_parent - sub_box.height;
        final_children.push(Child {
            offset: Point { x: b.width, y: sub_top },
            child: sub_box,
        });
    }

    Box {
        width: max_w,
        height: parent_height,
        depth: parent_depth,
        kind: BoxKind::HBox(final_children),
    }
}

```rust
//! Box tree → SVG byte stream.

use std::fmt::Write;

use crate::boxer::{Box, BoxKind, Child, Point};
use crate::font;

pub fn emit(root: &Box) -> Vec<u8> {
    let w = root.width.max(0.0);
    let h = (root.height + root.depth).max(0.0);
    let mut out = String::new();
    let _ = write!(
        &mut out,
        r#"<svg xmlns="http://www.w3.org/2000/svg" width="{w}" height="{h}" viewBox="0 0 {w} {h}">"#,
        w = w, h = h
    );
    walk(&mut out, root, Point { x: 0.0, y: 0.0 });
    out.push_str("</svg>");
    out.into_bytes()
}

fn walk(out: &mut String, b: &Box, origin: Point) {
    match &b.kind {
        BoxKind::Glyph { glyph_id, font_size } => {
            let s = font_size / font::units_per_em();
            let path_d = font::outline_path(*glyph_id);
            if path_d.is_empty() { return; }
            // Baseline of this glyph sits at y = origin.y + b.height
            let baseline_y = origin.y + b.height;
            // matrix(s 0 0 -s ox oy) — y-flip
            let _ = write!(
                out,
                r#"<path transform="matrix({s} 0 0 {neg_s} {ox} {oy})" d="{d}"/>"#,
                s = s, neg_s = -s, ox = origin.x, oy = baseline_y, d = path_d
            );
        }
        BoxKind::Rule { thickness } => {
            let _ = write!(
                out,
                r#"<rect x="{x}" y="{y}" width="{w}" height="{h}"/>"#,
                x = origin.x, y = origin.y, w = b.width, h = thickness
            );
        }
        BoxKind::HBox(children) | BoxKind::VBox(children) => {
            for Child { offset, child } in children {
                walk(out, child, Point { x: origin.x + offset.x, y: origin.y + offset.y });
            }
        }
        BoxKind::Empty => {}
    }
}
```

fn rasterize_svg(svg_bytes: &[u8], scale: f32) -> Vec<u8> {
    let opts = resvg::usvg::Options::default();
    let tree = resvg::usvg::Tree::from_data(svg_bytes, &opts).expect("usvg parse");
    let size = tree.size();
    let w = (size.width() * scale).ceil() as u32;
    let h = (size.height() * scale).ceil() as u32;
    let mut pixmap = tiny_skia::Pixmap::new(w.max(1), h.max(1)).unwrap();
    let ts = tiny_skia::Transform::from_scale(scale, scale);
    resvg::render(&tree, ts, &mut pixmap.as_mut());
    pixmap.encode_png().unwrap()
}

#[test]
fn pixel_snapshots() {
    insta::glob!("corpus/*.tex", |path| {
        let src = fs::read_to_string(path).unwrap();
        let src = src.trim();
        let style = if src.contains(r"\sum") || src.contains(r"\int") || path.to_string_lossy().contains("display") {

        let b = boxer::layout(&ir, style);
        let svg_bytes = svg::emit(&b);
        let png = rasterize_svg(&svg_bytes, 4.0);  // 4x for crisp comparison

    });
