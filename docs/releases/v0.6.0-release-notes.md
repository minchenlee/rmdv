# rmdv v0.6.0 — packaged CLI, file associations, and clearer rendering

> Release notes prepared 2026-07-21. Publication is still pending.
>
> Source: `main@4311fbf`; release candidate: `codex/release-v0.6.0` (2026-07-21).

rmdv v0.6.0 makes the native app easier to install and use from the desktop,
Finder, and Terminal. It also improves CJK rendering and aligns the public
site with the current product.

## Highlights

### Packaged CLI installation

- Install the `rmdv` command directly from the app's startup prompt or command
  palette on macOS and Linux.
- macOS installs `/usr/local/bin/rmdv` with administrator authorization when
  needed.
- Linux installs a user-scoped `~/.local/bin/rmdv` symlink to the AppImage.
- The CLI continues to use the existing local IPC interface for opening files,
  navigation, view changes, and state queries.

### Finder file associations

Packaged macOS builds register rmdv as an editor for Markdown, plain text,
structured text, and LaTeX files:

- `.md`, `.markdown`, `.mdown`, `.mkdn`, `.mkd`
- `.txt`, `.text`
- `.json`, `.yaml`, `.yml`, `.toml`
- `.tex`

### CJK rendering and mindmap hints

- CJK, Hangul, Japanese, and mixed-script runs now use a reliable fallback
  path without changing Latin styling.
- CJK emphasis adjacent to full-width punctuation remains correctly parsed.
- Document and Full Mindmap hint islands now expose the controls belonging to
  their own surface, including zoom where it is available.

### Public site and release packaging

- The landing page presents rmdv as a native product workspace with responsive
  documentation, real product captures, and the interactive AI disclosure
  sticker.
- The macOS release workflow builds the custom drag-to-Applications DMG and
  carries the packaged CLI/file-association metadata.
- Linux release builds refresh PDFium artifacts before packaging and fall back
  to the PDFium cache when the build tree does not contain the runtime library.

## Verification

- Local `cargo check`, both feature checks, `cargo test --lib`,
  `cargo test --tests`, and `cargo build --release --bin rmdv` passed on Apple
  Silicon.
- Targeted Apple Silicon release build and `cargo packager` app bundle build
  passed on the v0.6.0 release candidate.
- The release binary reports `rmdv 0.6.0`; the packaged app reports bundle
  version `0.6.0`.
- The packaged `Info.plist` contains the four expected file-association groups.
- Owner-reported manual acceptance passed for Finder associations, CLI reset and
  installation, CJK rendering, and native regression smoke.
- Linux CI passed the release binary, AppImage packaging, and CLI smoke checks
  for the merged PR.

## Platform notes

- macOS releases are signed and notarized when release secrets are present.
- Linux ships as an x86_64 AppImage with local PDF extraction enabled.
- Windows remains a best-effort build and uses `--no-default-features`; its
  package acceptance remains a separate cross-platform gate.
- Site publication requires the local authenticated Wrangler path and a live
  homepage plus `llms.txt` verification.
