## Core message

**Install once. Open from anywhere. See every script and document clearly.**

rmdv v0.6.0 extends the native Markdown viewer into the desktop workflow: the
packaged app can install its CLI, Finder can open supported text documents with
rmdv, and CJK content renders with dependable fallback behavior.

## User-facing copy

### CLI installation

After opening the packaged app for the first time, use the `Install CLI` toast
or run `Install CLI` from the command palette. On macOS this creates
`/usr/local/bin/rmdv`; on Linux it creates `~/.local/bin/rmdv`.

### File associations

The macOS app is an editor association for Markdown, plain text, JSON/YAML/TOML,
and LaTeX documents. The package must be installed into `/Applications` before
Finder Open With is manually checked.
