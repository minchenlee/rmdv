## Core message

**See the shape of a workspace.**

rmdv is a native Rust Markdown viewer that now lets people move from a single
document to the structure of an entire folder without leaving the keyboard.
Full Mindmap Mode is an opt-in workspace navigator; the existing document
Mindmap remains the fast way to inspect one file.

```text

```

### FAQ addition

```text
What is the difference between document Mindmap and Full Mindmap?

Document Mindmap (`⌘M`) turns the current Markdown, JSON, or YAML document into
a tree. Full Mindmap (`⌘⇧M`) is an app-level workspace navigator: it shows the
folders and files in a workspace and lets you preview or open a selected file.
The two surfaces are separate, so entering one does not overwrite the other.
```
