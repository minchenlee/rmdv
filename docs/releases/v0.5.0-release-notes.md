# rmdv v0.5.0 — Full Mindmap workspaces and smoother navigation

> Release notes for v0.5.0, published 2026-07-18.
>
> Source: `main@33b7d8f` (2026-07-18).

rmdv v0.5.0 makes Markdown workspaces easier to see and move through. It adds
an opt-in Full Mindmap workspace navigator, keeps document mindmaps focused and
independent, and makes graph navigation feel natural with animated zoom input.

## Highlights

### Full Mindmap Mode

Full Mindmap Mode turns a folder into a navigable workspace graph. It is
separate from the existing document Mindmap view, so opening it does not
replace or disturb the document currently being read or edited.

- Enter Full Mindmap Mode with `⌘⇧M`.
- Use the arrow keys to move through workspace nodes and `Space` to expand or
  collapse the current branch.
- Select a file to preview it in place; press `Enter` to open it through the
  normal document-loading path.
- Return to the previous surface with `Esc` or `⌘⇧M`.
- Use `⌘⌥W` to cycle the preview panel through its supported widths.
- Navigate between the workspace graph and the existing document Mindmap
  without losing the selected file or focus target.

Workspace indexing, file previews, and asset loading are bounded and guarded
against stale background results, keeping navigation responsive on large
folders and long Markdown files.

### Native mindmap zoom

Both document and Full Mindmap canvases support smooth, anchor-preserving zoom:

- `=` / `+` zoom in.
- `-` zoom out.
- `0` resets the graph zoom.
- Mouse-wheel, touch pinch, and macOS trackpad pinch zoom around the input
  position.

The existing document typography shortcuts remain separate from graph zoom.

### Zen editing and safer navigation

Edit mode is now a focused Zen surface with unsaved-edit protection. Moving
between files, opening a workspace node, or switching views keeps the dirty-file
guard in place instead of silently discarding edits.

## Also included

- More reliable Full Mindmap selection, focus, preview, and exit transitions.
- Virtualized long Markdown previews with background parsing and stale-result
  protection.
- More predictable panel visibility: opening Full Mindmap starts with the panel
  closed, and navigation does not override the user's panel choice.
- The existing native Markdown, Mermaid, Graphviz, LaTeX, PDF, search, themes,
  live reload, IPC, and auto-update features remain available.

## Platform notes

- macOS releases are signed and notarized when the release secrets are present.
- Linux ships as an x86_64 AppImage with local PDF extraction enabled.
- Windows remains a best-effort build and uses `--no-default-features`; PDF
  extraction is currently a macOS/Linux feature.
